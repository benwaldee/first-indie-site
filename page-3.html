How to use your codes:

Congratulations! You have obtained the 3 codes. The first number of the code is the website link you need to take. The second number is the header number, the third number is the letter number. Note that the numbers do not go down to 0 (this is not an index, this is a game), so the number 1 could indicate the first letter of a header. Spaces do NOT count as letters. Assuming link 3 went to the current page, the code 318 would yield an "e". Once you are finished, remember your 3 letters and their order and move on to page 4! (HINT: it should spell an animal)
